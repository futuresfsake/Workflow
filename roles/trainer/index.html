Review Student Profiles
1
Navigate to Active Student Directory View
From your active class selection layout, scroll downward to locate the integrated relational student grid mapping.
2
Parse Unique Learner Identifier (ULI) Records
Verify each trainee's physical credentials against their system-rendered national TESDA ULI string parameters or their personal ID in the system.
Possible Warning For Step If a trainee's registration metadata displays an unverified status label, contact the master Registrar administrative office to parse missing archival records before saving progress milestones.
